# F1 official drinking game

## Rules

* Sip - 16ml +
* Shot - 55ml +

## Sips

1. Two sips for an unsafe release
2. One sip for a brake lock-up

## Shots

1. Two shots if a car has issues during the formation lap
2. One shot for a big accident


## Finish your drink

1. Red flag
2. Latifi gets in the top 10
3. Hamilton DNF
